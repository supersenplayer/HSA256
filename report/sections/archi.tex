\section{Architecture materielle}
\label{sec:archi}

\subsection{Choix architectural}

Architecture \textbf{iterative} : une seule instance du circuit de ronde, reutilisee 64 fois (1 ronde/cycle). Compromis optimal surface/debit pour un co-processeur embarque.

\textbf{Protocole} : le hote ecrit num\_blocks a 0x0000 + les blocs pre-paddes a partir de 0x0004, puis pulse \texttt{start}. Le co-processeur traite tous les blocs et ecrit le condense a 0x2000. Le signal \texttt{done} indique la fin.
