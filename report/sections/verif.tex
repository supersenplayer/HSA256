\section{Verification et resultats}
\label{sec:verif}

\subsection{Strategie}

Banc de test auto-verifiant : pre-charge message $\rightarrow$ pulse start $\rightarrow$ capture writeback $\rightarrow$ compare au reference (Python \texttt{hashlib}).
